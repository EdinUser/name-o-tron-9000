# Glossary

## Plex Metadata

The titles, years, seasons, episodes, album data, artwork references, and provider identifiers that Plex has matched to your media.

## Provider ID

A stable identifier from an external database, such as IMDb, TMDb, TVDB, or MusicBrainz. Name-o-Tron can preserve or append provider IDs when Plex exposes them.

## Media Root

The top folder of a library, such as `/media/Movies` or `D:\Media\TV`.

## Path Mapping

The rule that maps a path seen by Plex to a path visible on the computer running Name-o-Tron.

## Sidecar File

A file stored next to a media file, such as an external subtitle, image, `.nfo`, or text file.

## Subtitle Sidecar

An external subtitle file such as `.srt`, `.ass`, `.ssa`, or `.vtt`.

## NFO

A metadata sidecar format used by some media tools, especially Kodi. Name-o-Tron does not currently generate Kodi-compatible `.nfo` files.

## Edition

A content variant such as Director's Cut, Extended Edition, Theatrical Cut, Final Cut, or Remastered.

## Version

A technical or release variant of the same media item, such as a different resolution, encode, or source.

## Multi-Episode File

A single video file that contains more than one TV episode, such as `S01E01-E02`.

## Library Normalization

The process of turning matched metadata into consistent filenames, folders, subtitles, IDs, and safety-checked filesystem operations.
